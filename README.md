## BeatBox

* 欲しい機能

8beatのみ2小説のループ
音色は8個
+ クラッシュシンバル
+ ハイハット（クローズ）
+ ライドシンバル
+ スネア
+ Hiタム
+ Loタム
+ フロアタム
+ バスドラム
  
特定の流せるマークした箇所のみ音が鳴る
ループで再生
速度変更可能
